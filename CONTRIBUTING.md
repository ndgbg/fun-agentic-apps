# Contributing to Fun Agentic Apps

Thanks for your interest in contributing! This repository showcases autonomous AI agents built to solve real-world problems.

## How to Contribute

### Adding Your Own Agent

Have an agentic app you'd like to add? Here's how:

1. Fork this repository
2. Create a new folder for your agent: `your-agent-name/`
3. Include:
   - Complete source code
   - README.md with description and setup instructions
   - Demo or screenshots
4. Update the main README.md to list your agent
5. Submit a pull request

### Improving Existing Agents

- Bug fixes
- Feature enhancements
- Documentation improvements
- Performance optimizations

## Guidelines

- Keep code clean and well-documented
- Include setup instructions
- Add a demo or screenshots
- Explain what makes your agent "agentic" (autonomous, proactive, goal-oriented)

## Questions?

Open an issue or reach out via GitHub discussions!
